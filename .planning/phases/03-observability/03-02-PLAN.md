<!-- Backend API contract from Plan 03-01 that this plan consumes -->

Dashboard endpoints (all require Bearer token, viewer+ role):
- GET /api/v1/dashboard/metrics -> { total_users: int, active_sessions: int, nas_count: int, recent_auth_failures: int }
- GET /api/v1/dashboard/auth-rates?range=24h -> [{ bucket: string, success: int, failure: int }]
- GET /api/v1/dashboard/traffic-per-nas -> [{ nas_ip: string, shortname: string|null, bytes_in: int, bytes_out: int }]
- GET /api/v1/dashboard/top-users?by=traffic&limit=10 -> [{ username: string, total_bytes: int, total_session_time: int }]
- GET /api/v1/dashboard/sessions/stream?token=xxx -> SSE: data: {"active_sessions": N}

Log endpoints (all require Bearer token, viewer+ role):
- GET /api/v1/logs/accounting?username=&nas_ip=&date_from=&date_to=&page=1&page_size=20 -> PaginatedResponse<AccountingRecord>
  AccountingRecord: { radacctid, username, nas_ip_address, framedipaddress, acct_start_time, acct_stop_time, acct_session_time, acct_input_octets, acct_output_octets, terminate_cause }
- GET /api/v1/logs/sessions?page=1&page_size=20 -> PaginatedResponse<ActiveSession>
  ActiveSession: { radacctid, username, nas_ip_address, framedipaddress, acct_start_time, acct_session_time, acct_input_octets, acct_output_octets }
- GET /api/v1/logs/postauth?username=&status=accept|reject&date_from=&date_to=&page=1&page_size=20 -> PaginatedResponse<PostAuthRecord>
  PostAuthRecord: { id, username, reply, authdate }

Existing frontend patterns:
- apiFetch(path, options) from '@/lib/api' — adds Bearer token, auto-refresh on 401
- getAccessToken() from '@/lib/api' — returns current access token string
- buildQuery(params) pattern for query string construction
- parseJson<T>(res) pattern for response parsing
- DataTable<T> component with columns, data, total, page, pageSize, onPageChange, onSearch, rowKey props
- useQuery from @tanstack/react-query for data fetching
- formatDuration(seconds) and formatBytes(octets) helpers from sessions page
- Intl.DateTimeFormat('fr-FR') for French date formatting
- Base UI Button with render prop for link navigation
